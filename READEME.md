🔗 URL Shortener & Analytics Dashboard

📌 Project Overview

This project is a Flask-based URL Shortener with an Analytics Dashboard. It allows users to shorten long URLs, generate QR codes, track clicks, analyze visitor data, and manage URLs through an easy-to-use web interface.

---

🚀 Features

- URL Shortening
- Custom Short Codes
- Click Tracking
- Analytics Dashboard
- Dark Mode
- QR Code Generation
- QR Code Download
- CSV Export
- PDF Export
- Bulk URL Upload (CSV)
- Edit URL
- Delete URL
- URL Expiry System
- Top 5 Most Clicked URLs
- Device Analytics
- Recent Visit History
- Search URLs
- Active & Expired URL Filter

⚙ Installation

1. Clone the repository

git clone <repository_link>

2. Open the project folder

cd URL_shortner_project

3. Install dependencies

pip install -r requirements.txt

4. Run the project

python app.py

5. Open the browser

http://127.0.0.1:5000
